# Slides Tools

MCP tool server for PowerPoint slide generation.

## Files

| File | Description |
|------|-------------|
| `exec.py` | MCP server providing `execute_code` for PPTX generation |

## Tools

### exec.py

- `execute_code` - Execute python-pptx code to generate slides

## Environment

Requires `pptx` conda environment (Python 3.10).

## Dependencies

- python-pptx
- LibreOffice (for PPTX to image conversion)
